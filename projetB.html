Euro Prevision

Le projet "Euro Prevision" consiste à constituer une base de donnée PSQL contenant tout les resultats du loto depuis 2014, puis avec un programme java constituer des données tels que les numeros qui sortent le plus souvant ceux qui ne sont pas sortis depuis longtemps ou encore le temps moyen entre deux sortie d'un numero. Sans oublier un algorithme de Machine learning qui essaye de prevoir les resultats pour une date donnée.

Compétences techniques visées

La programmation en Java
Creation de zero d'une base PSQL
L'intégration de machine learning pour tenter de prevoir les resultats

Savoirs êtres

Autonomie et curiosité :

Capacité à explorer et implémenter des concepts avancés tels que le machine learning pour enrichir la simulation et à expérimenter avec des approches nouvelles pour résoudre des problèmes complexes.

Résolution de problèmes et innovation :

Compétences en résolution de problèmes techniques et en recherche de solutions créatives pour intégrer efficacement le machine learning dans un système déjà complexe.
