SAÉ 2.01 + 2.02 dev d’une application et exploration algorithmique

La SAE 2.02, consacrée au développement d'une application et à l'exploration algorithmique, a pour but d'apprendre à utiliser une bibliothèque spécifique pour créer une application répondant à des critères techniques précis. Le projet consistait à développer une application capable de calculer les trajets les plus courts entre deux villes en tenant compte de plusieurs modes de transport comme le train ou l'avion, et en évaluant divers critères tels que le prix, la durée et les émissions de CO2. Pour ce faire, nous avons utilisé une bibliothèque fournie pour créer un graphe, tout en nous chargeant également de la partie interface utilisateur (IHM). Nous avons conçu une application visuelle permettant à l'utilisateur de saisir les informations nécessaires et de recevoir les trajets correspondants.

Compétences techniques visées

La programmation en Java
L’utilisation de bibliothèques externes
Le prototypage avec Figma
Le développement avec JavaFX

Savoirs êtres

Autonomie et initiative :

Aptitude à travailler de manière autonome pour explorer et maîtriser la bibliothèque fournie, ainsi que pour résoudre les problèmes techniques qui peuvent survenir.

Collaboration et communication :

Compétences en communication pour discuter des choix techniques, partager les progrès, et résoudre les éventuels désaccords de manière constructive.

Projet Agile : Jeu d'échecs

Le projet Agile consistait à créer un jeu en Java à utiliser sur terminal, avec une équipe de six personnes, en s'appuyant sur la méthodologie Agile. Nous avons choisi de développer un jeu d'échecs, en nous organisant en sprints de 2 heures. L'objectif était de concevoir un jeu fonctionnel permettant à deux joueurs de s'affronter, tout en respectant les règles classiques des échecs. Nous avons ainsi mis en place une architecture de classes pour gérer le plateau, les pièces, les mouvements et les règles, tout en veillant à ce que le jeu soit fluide et intuitif à jouer via une interface terminal.

Compétences techniques visées

La programmation en Java
L'implémentation d'un jeu avec gestion des règles
La gestion de projet Agile en sprints
Le travail collaboratif en équipe

Savoirs êtres

Agilité et gestion du temps :

Capacité à organiser le travail en sprints de 2 heures, en définissant des objectifs clairs et en réajustant le travail à chaque itération pour atteindre un produit final fonctionnel.

Collaboration et communication :

Compétences en travail d'équipe et en communication pour coordonner les tâches entre les membres, partager les progrès et résoudre les défis rencontrés durant le développement.

Primordial Soup : Application Java avec Machine Learning

Le projet "Primordial Soup" consiste à développer une application Java qui est une version avancée du jeu de la vie de Conway, inspirée de l’approche de Lenya. Cette application simule un environnement où des cellules évoluent selon des règles spécifiques, mais avec l'ajout du machine learning pour permettre à ces cellules de s’adapter et de trouver des configurations intéressantes de manière autonome. L'objectif était d'utiliser le machine learning pour générer et évaluer différentes configurations, permettant aux cellules de "s'auto-optimiser" et d'évoluer vers des résultats intéressants sans intervention humaine directe.

Compétences techniques visées

La programmation en Java
L'implémentation du jeu de la vie de Conway
L'intégration de machine learning pour l'auto-optimisation
La simulation et modélisation d'environnements dynamiques

Savoirs êtres

Autonomie et curiosité :

Capacité à explorer et implémenter des concepts avancés tels que le machine learning pour enrichir la simulation et à expérimenter avec des approches nouvelles pour résoudre des problèmes complexes.

Résolution de problèmes et innovation :

Compétences en résolution de problèmes techniques et en recherche de solutions créatives pour intégrer efficacement le machine learning dans un système déjà complexe.
