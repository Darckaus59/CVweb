Primordial Soup : Application Java avec Machine Learning

Le projet "Primordial Soup" consiste à développer une application Java qui est une version avancée du jeu de la vie de Conway, inspirée de l’approche de Lenya. Cette application simule un environnement où des cellules évoluent selon des règles spécifiques, mais avec l'ajout du machine learning pour permettre à ces cellules de s’adapter et de trouver des configurations intéressantes de manière autonome. L'objectif était d'utiliser le machine learning pour générer et évaluer différentes configurations, permettant aux cellules de "s'auto-optimiser" et d'évoluer vers des résultats intéressants sans intervention humaine directe.

Compétences techniques visées

La programmation en Java
L'implémentation du jeu de la vie de Conway
L'intégration de machine learning pour l'auto-optimisation
La simulation et modélisation d'environnements dynamiques

Savoirs êtres

Autonomie et curiosité :

Capacité à explorer et implémenter des concepts avancés tels que le machine learning pour enrichir la simulation et à expérimenter avec des approches nouvelles pour résoudre des problèmes complexes.

Résolution de problèmes et innovation :

Compétences en résolution de problèmes techniques et en recherche de solutions créatives pour intégrer efficacement le machine learning dans un système déjà complexe.
